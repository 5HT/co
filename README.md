# N2O: Coq Application Server

Only pure console infinite I/O here for now, stay tuned.

## Prerequisites

    opam repo add coq-released https://coq.inria.fr/opam/released
    opam install coq-io-system

## Extract

    ./configure.sh
    make

## Build and Run

    cd extraction/
    make
    ./main.native
